# Rent A Car Project - Front End 

Kodlama.io - Software Bootcamp.


### Technologies  

* Visual Studio Code  
* Angular 
* Bootstrap v5.0.0-beta2
* Node.js v14.16.0


```bash
npm install -g @angular/cli    # to install Angular CLI  
ng serve --open                # to launch the server, rebuild the app
```
